To run the game, you go into the build folder and run the Entryway.exe file.

Movement keys are: W for forward, A for moving left, S for moving backwards, D for moving right.
To turn you press: Q for left turning and E for right turning. 
To jump you press: Space and to double jump you press: Space again.
To dash you press: left shift.
To shoot you press: left mouse button.

If you run out of mana to shoot, you pick up the crystals.
